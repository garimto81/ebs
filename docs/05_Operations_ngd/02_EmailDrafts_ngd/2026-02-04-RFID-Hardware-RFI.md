# [보류] RFID Hardware 업체 RFI

**상태**: 보류 (Phase 1+)
**보류 사유**: 카테고리 B (부품 공급). 리더 모듈만 공급하며 RFID 카드 역량 없음.
**대상**: FEIG, GAO RFID, Identiv, PONGEE
**재활용 시기**: Phase 1에서 리더 모듈 별도 조달이 필요할 때
