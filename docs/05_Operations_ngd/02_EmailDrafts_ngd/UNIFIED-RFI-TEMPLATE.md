## 영문 템플릿

**Subject**: Partnership Inquiry - RFID Card Recognition System for Live Poker Broadcasting

---

Hi,

I'm developing a real-time card recognition system for live poker broadcasting. The system needs to automatically identify each playing card as it's dealt and display the information on the broadcast screen instantly.

I'm looking for a partner who can provide an integrated solution covering both RFID-embedded playing cards and the reader hardware.

**What we need:**

1. RFID Playing Cards
   - Standard poker size (63 x 88mm), RFID tag embedded in each card
   - 54 cards per deck (52 + 2 jokers), each uniquely identifiable
   - Durable enough for professional use (repeated handling over multi-hour sessions)

2. RFID Reader Hardware
   - Compact form factor suitable for embedding into a poker table surface
   - Capable of identifying a single card within 200ms
   - Stable continuous operation for 4+ hours
   - Software integration interface (SDK, API, or protocol documentation)

3. Integration Support
   - Our team handles the software integration
   - We need the hardware to provide card identification data that our software can consume
   - Development documentation and technical support during integration

**Questions:**

1. Can you supply both the RFID cards and the reader hardware, or only one of them?
2. Do you have experience with similar card recognition or RFID identification projects?
3. Is it possible to start with a small pilot order (e.g., 5-10 reader units and 50-100 card decks) for evaluation?
4. What would be the approximate timeline and cost range for a pilot?
5. Are you able to customize the solution to our specific requirements?

I'd appreciate any relevant product information, case studies, or a follow-up call to discuss further.

Thank you for your time.

Best regards,
Aiden

## 국문 템플릿 (국내 업체용)

**제목**: 파트너십 문의 - 포커 방송용 RFID 카드 인식 시스템

---

안녕하세요,

실시간 포커 방송에서 카드를 자동으로 인식해 방송 화면에 표시하는 시스템을 개발하고 있습니다.

RFID가 내장된 포커 카드와 리더 하드웨어를 통합으로 제공할 수 있는 파트너를 찾고 있습니다.

**필요 사항:**

1. RFID 포커 카드
   - 표준 포커 카드 크기 (63 x 88mm), 카드마다 RFID 태그 내장
   - 덱당 54장 (52장 + 조커 2장), 개별 식별 가능
   - 프로 환경에서의 내구성 (수 시간 반복 핸들링)

2. RFID 리더 하드웨어
   - 포커 테이블 표면에 매립 가능한 소형 폼팩터
   - 카드 1장을 200ms 이내에 인식
   - 4시간 이상 연속 안정 운영
   - 소프트웨어 연동 인터페이스 (SDK, API 또는 프로토콜 문서)

3. 연동 지원
   - 소프트웨어 연동은 자체 개발
   - 하드웨어에서 카드 식별 데이터를 제공하면 소프트웨어가 처리하는 구조
   - 연동 개발을 위한 기술 문서 및 지원

**문의사항:**

1. RFID 카드와 리더 하드웨어를 모두 공급 가능하신가요, 아니면 한쪽만 가능하신가요?
2. 유사한 카드 인식 또는 RFID 시스템 프로젝트 경험이 있으신가요?
3. 소규모 파일럿 주문 (리더 5-10대, 카드 50-100덱)으로 시작할 수 있나요?
4. 파일럿 기준 대략적인 일정과 비용 범위는 어떻게 되나요?
5. 요구사항에 맞는 커스텀 개발이 가능하신가요?

관련 제품 정보, 사례, 또는 상세 논의를 위한 통화 일정을 주시면 감사하겠습니다.

감사합니다.
Aiden
