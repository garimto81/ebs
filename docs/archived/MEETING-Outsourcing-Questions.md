# PokerCaster 미팅 질문서

**미팅 일자**: 2026-01-26
**미팅 성격**: 첫 미팅 / 솔루션 탐색

---

## PokerCaster 배경 정보

PokerCaster는 EPT(European Poker Tour) 및 WSOPE(World Series of Poker Europe) 방송에 참여한 경력이 있는 팀입니다.

> **인정**: 이러한 경력은 포커 방송 도메인에 대한 실질적인 경험을 보여줍니다.
> 따라서 "RFID 경험이 있는가?" 같은 기초적 검증 질문은 불필요합니다.

---

## 핵심 질문 8개

### Q1. 현재 개발 중인 시스템은 어느 단계인가요?
**EN: What stage is your system currently at?**

| 단계 | 의미 |
|------|------|
| 컨셉 | 아이디어/설계 단계 |
| 프로토타입 | 동작하는 샘플 있음 |
| 베타 | 테스트 운영 중 |
| 상용 | 실제 서비스 중 |

> **왜 필요한가**: 프로덕트 성숙도 파악. 단계에 따라 협업 방식(기술 지원 vs 공동 개발 vs 솔루션 구매)이 달라짐

---

### Q2. 자체 개발인지, 기존 솔루션(PokerGFX 등) 기반인지?
**EN: Is it self-developed or based on existing solutions like PokerGFX?**

> **왜 필요한가**: 기술 독립성 확인. 자체 개발이면 커스터마이징 자유도 높음, 기존 솔루션 기반이면 라이선스 이슈 검토 필요

---

### Q3. RFID 하드웨어도 직접 개발하셨나요?
**EN: Did you develop the RFID hardware yourself?**

> **왜 필요한가**: 하드웨어 공급 가능 여부 파악. 직접 개발이면 턴키 솔루션(HW+SW) 가능, 아니면 별도 하드웨어 조달 필요

---

### Q4. 현재 구현된 주요 기능은 어떤 것들이 있나요?
**EN: What main features are currently implemented?**

우리가 필요한 기능 체크리스트:

| 기능 | 필수 | 있음? |
|------|:----:|:-----:|
| 홀카드 인식 | O | |
| 커뮤니티 카드 인식 | O | |
| 팟 사이즈 표시 | O | |
| 승률 계산 | O | |
| Secure Delay | O | |
| 멀티 테이블 | - | |

> **왜 필요한가**: 기능 범위 파악. 우리가 필요한 기능이 이미 구현되어 있는지, 추가 개발이 필요한지 확인

---

### Q5. 오버레이 그래픽은 어떤 방식으로 처리하시나요?
**EN: How do you handle overlay graphics?**

| 방식 | 특징 |
|------|------|
| OBS 브라우저 소스 | 웹 기반, 연동 쉬움 |
| NDI | 네트워크 비디오, 전문 장비용 |
| 독립 렌더링 | 별도 출력, 캡처 필요 |
| vMix/Wirecast 연동 | 방송 소프트웨어 플러그인 |

> **왜 필요한가**: 기술 스택 이해. 우리 방송 환경(OBS 기반)과의 호환성 판단

---

### Q6. 실제 서비스 오픈 예정은 언제쯤인가요?
**EN: When is your planned service launch?**

> **왜 필요한가**: 타임라인 파악. 우리 일정과 맞는지, 완성된 솔루션을 바로 쓸 수 있는지, 개발 참여가 필요한지 판단

---

### Q7. PokerGFX와 비교해서 차별점이 있다면?
**EN: What differentiates you from PokerGFX?**

참고: PokerGFX 특징
- 업계 표준 솔루션
- 고가 (수천만원 추정)
- 독점적 하드웨어

> **왜 필요한가**: 경쟁력 이해. 왜 이 솔루션을 선택해야 하는지, 가격/기능/유연성 측면에서 기존 대비 장점이 무엇인지 파악

---

### Q8. 프로덕트 데모를 볼 수 있을까요?
**EN: Can we see a product demo?**

> **왜 필요한가**: 다음 단계 설정. 말로 설명 듣는 것과 실제 동작을 보는 것은 다름. 시연을 통해 실제 완성도 확인. 자연스러운 후속 미팅 설정
