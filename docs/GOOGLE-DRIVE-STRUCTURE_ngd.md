## 관리 규칙

| 규칙 | 내용 |
|------|------|
| **단일 계층** | 하위 폴더 없음, EBS root에 직접 배치 |
| **문서 수** | 항상 2개 유지 |
| **동기화 도구** | `python -m lib.google_docs convert` |
| **이름 변경 도구** | `python ebs/rename_ebs_docs.py` (Drive 표시 이름 PRD 번호 체계 적용) |
| **매핑 파일** | `docs/MAPPING_ngd.json` v9.0.0 |
| **_ngd 접미사** | 이 파일 자체는 Drive 비공유 메타 문서 |
