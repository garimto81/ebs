## 검증 체크리스트

구매 전 다음을 확인하세요:

### ESP32 확인사항
- [ ] ESP32-WROOM-32 또는 ESP32-DevKitC 모델
- [ ] 30핀 또는 38핀 (배선 가이드와 일치)
- [ ] USB-C 또는 Micro USB 포트 확인
- [ ] 3.3V/5V 출력 핀 존재

### MFRC522 확인사항
- [ ] 13.56MHz 주파수
- [ ] SPI 인터페이스 지원
- [ ] 8핀 헤더 (SDA, SCK, MOSI, MISO, IRQ, GND, RST, VCC)
- [ ] 안테나 일체형 (외장 안테나 모델 주의)

### RFID 카드 확인사항
- [ ] MIFARE Classic 1K (S50) 또는 4K (S70)
- [ ] 13.56MHz ISO14443A 표준
- [ ] UID 읽기 가능 여부
- [ ] **⚠️ MIFARE Ultralight는 호환 안됨**
