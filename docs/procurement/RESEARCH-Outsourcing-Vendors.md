## 1. Executive Summary

EBS 프로젝트의 RFID 하드웨어 개발을 위한 외주 업체 선정 리서치 결과입니다. ST25R3911B 기반 프로덕션 모듈 제작을 위해 국내외 PCB 제조, RFID 전문, EMS 업체를 조사했습니다.

**핵심 결론**:
- **Stage 0 프로토타입**: JLCPCB (중국) - 최저 비용, 빠른 프로토타입
- **Stage 1+ 프로덕션**: KOREAECM 또는 SIMMTECH (한국) - 고품질, 현지 서비스
- **대안**: 하이브리드 전략 (중국 프로토타입 → 한국 양산)

## 5. 추천 전략

### 5.1 Stage별 권장 업체

```
Stage 0 (현재): RFID 연결 검증
    ├── Phase 0-1: MFRC522 학습
    │   └── 업체: 알리익스프레스 (MFRC522 $0.97)
    │
    ├── Phase 0-2: E2E PoC
    │   └── 업체: 알리익스프레스 (ST25R3911B 개발키트 $50)
    │
    ├── Phase 0-3: 프로덕션 스펙 검증
    │   └── 업체: JLCPCB (커스텀 PCB + ST25R3911B IC)
    │
    └── Gate: 프로덕션 스펙 충족 확인
        └── 비용: $100-$150 (프로토타입 3-5개)

Stage 1: PokerGFX 클론
    ├── 소량 양산 (10-20개)
    │   └── 업체: KOREAECM 또는 JLCPCB
    │
    └── 안테나 최적화
        └── 업체: PCBWay (안테나 설계 상담) 또는 엠포플러스 (컨설팅)

Stage 2+: WSOP 통합
    ├── 중량 양산 (50-200개)
    │   └── 업체: SIMMTECH 또는 KOREAECM
    │
    └── 인증/품질관리
        └── 업체: LG Innotek (RF 인증 필요 시)
```

## 6. 즉시 실행 가능한 액션 아이템

### 6.1 우선순위 1: Stage 0 프로토타입 (1-2주 내)

- [ ] **JLCPCB 계정 생성** (jlcpcb.com)
- [ ] **Gerber 파일 준비** (KiCad 또는 EAGLE로 PCB 설계)
- [ ] **BOM 리스트 작성** (ST25R3911B-AQFT 포함)
- [ ] **온라인 견적 확인** (예상 비용 $50-$100)
- [ ] **프로토타입 3개 주문** (ESP32 + ST25R3911B)

**병행 작업**:
- [ ] **알리익스프레스 주문** (MFRC522 모듈 $0.97, ST25R3911B 개발키트 $50)
- [ ] 즉시 학습 시작 (MFRC522로 Arduino 테스트)

### 6.2 우선순위 2: 국내 업체 컨택 (2-4주 내)

- [ ] **KOREAECM 문의 메일**
  - 첨부: BOM 리스트, 프로젝트 개요
  - 질문: 소량 주문 가능 여부, 안테나 설계 지원, 견적

- [ ] **엠포플러스 기술 상담**
  - 질문: ST25R3911B 취급 여부, 포커 테이블용 안테나 설계
