## 업데이트 가이드

### 문서 상태 변경 시

1. **해당 테이블에서 문서 찾기**
2. **버전, 상태, 마지막 업데이트 수정**
3. **전체 요약 통계 업데이트**
4. **Git Commit**:
   ```bash
   git add docs/operations/DOCUMENT-STATUS.md
   git commit -m "docs(ops): update document status - [문서명] [변경사항]"
   ```

### 새 문서 추가 시

1. **해당 Stage 테이블에 행 추가**
2. **전체 요약 "총 문서 수" 증가**
3. **상태별 상세에 추가**

### 문서 아카이브 시

1. **원래 테이블에서 제거**
2. **Archived Documents 테이블에 추가**
3. **전체 요약 통계 조정**
