**Status**: Active

## 1. Executive Summary

EBS 프로젝트 전체 Stage(0-3)에 걸친 업체 관리 통합 시스템입니다. RFID 하드웨어 제조, 펌웨어 개발, 컨설팅, 인증 등 모든 외주 업체를 중앙에서 추적하고 평가합니다.

**핵심 목표:**
- 업체 상태 실시간 추적 (후보 → 계약 → 평가)
- Stage별 업체 매핑 및 예산 관리
- 커뮤니케이션 이력 관리
- 평가 기준 표준화

## 2. 업체 대시보드

### 2.1 현재 업체 현황

| 업체명 | 카테고리 | Stage | 상태 | 담당자 | 최근소통일 | 평점 | 비고 |
|--------|----------|-------|------|--------|-----------|------|------|
| **JLCPCB** | PCB 제조 | 0 | 후보 | - | - | - | Stage 0 최우선 추천 |
| **KOREAECM** | EMS | 1-2 | 후보 | - | - | - | 소량 양산 적합 |
| **PCBWay** | PCB 제조 | 0 | 대안 | - | - | - | 안테나 설계 지원 |
| **SIMMTECH** | PCB 제조 | 2-3 | 후보 | - | - | - | 대량 양산 적합 |
| **엠포플러스** | RFID 컨설팅 | 0-1 | 후보 | - | - | - | 기술 자문 |
| **RFIOT** | RFID 전문 | 1 | 보류 | - | - | - | 커스텀 개발 미확인 |
| **에일리언테크** | RFID 하드웨어 | 2 | 보류 | - | - | - | 대형 프로젝트 위주 |
| **BK Electronics** | EMS | 1 | 후보 | - | - | - | 펌웨어 개발 가능 |

**상태 정의:**
- **후보**: 리서치 완료, 컨택 전
- **견적요청**: 문의 메일 발송, 응답 대기
- **협상중**: 견적 받음, 조건 협상
- **계약완료**: 계약 체결, 작업 진행
- **보류**: 현재 Stage 부적합
- **거절**: 조건 불일치, 탈락

### 2.2 Stage별 업체 맵핑

```
Stage 0 (프로토타입 검증)
├─ RFID 학습
│  └─ 알리익스프레스 (MFRC522 $0.97) [직접 구매]
│
├─ 프로토타입 제작
│  ├─ JLCPCB (1순위) - $100, 10-14일
│  └─ PCBWay (2순위) - $150, 15-22일
│
└─ 기술 컨설팅
   └─ 엠포플러스 (RFID 안테나 설계)

Stage 1 (PokerGFX 클론, 소량 양산 10-20개)
├─ 소량 양산
│  ├─ KOREAECM (1순위) - 한국, 빠른 대응
│  └─ JLCPCB (2순위) - 저렴, 품질 검증됨
│
├─ 안테나 최적화
│  ├─ PCBWay (설계 상담)
│  └─ 엠포플러스 (컨설팅)
│
└─ 펌웨어 개발 (선택)
   └─ BK Electronics (OEM/ODM)

Stage 2 (WSOP 통합, 중량 양산 50-200개)
├─ 중량 양산
│  ├─ SIMMTECH (1순위) - 글로벌 품질
│  └─ KOREAECM (2순위) - 검증된 파트너
│
├─ RFID 시스템 통합
│  └─ 에일리언테크놀로지 (대형 프로젝트 경험)
│
└─ 인증 (선택)
   └─ LG Innotek (RF 인증, EMC)

Stage 3 (방송 자동화, 대량 양산 200+개)
├─ 대량 양산
│  └─ SIMMTECH (품질 보증, 대량 인프라)
│
├─ 보안 인증
│  └─ 인증 전문 업체 (추후 선정)
│
└─ 유지보수
   └─ KOREAECM (현지 지원)
```

### 4.1 상태 전환 다이어그램

```

  │                        ├─ (견적 받음) → [협상중]
  │                        │                   │
```

## 5. 평가 기준 (5점 척도)

### 5.1 평가 항목 정의

| 기준 | 설명 | 측정 방법 |
|------|------|----------|
| **기술력** | 요구 스펙 충족, RFID 경험 | 샘플 품질, 안테나 설계 지원 |
| **가격** | 예산 대비 비용 효율 | 견적 비교 (경쟁사 대비) |
| **납기 준수** | 약속한 납기일 준수 | 실제 납기 vs 약속 납기 |
| **소통** | 대응 속도, 언어, 협조성 | 이메일 응답 시간, 미팅 질 |
| **품질** | 완제품 품질, 불량률 | QC 결과, 재작업 횟수 |

### 5.2 점수 기준

| 점수 | 평가 | 설명 |
|:----:|------|------|
| **5** | 탁월 | 기대 이상, 경쟁사 상위 10% |
| **4** | 우수 | 기대 충족, 만족스러움 |
| **3** | 보통 | 기대 충족, 표준 수준 |
| **2** | 미흡 | 기대 미달, 개선 필요 |
| **1** | 불량 | 기대 크게 미달, 재계약 불가 |

### 5.3 평가 시점

| Stage | 평가 시점 | 평가 방법 |
|-------|----------|----------|
| **Stage 0** | 프로토타입 납품 후 | 샘플 테스트 + 소통 평가 |
| **Stage 1** | 소량 양산 완료 후 | 품질 검수 + 납기 평가 |
| **Stage 2+** | 각 배치 완료 후 | 종합 평가 + 불량률 분석 |

### 5.4 가중 평가 (우선순위 반영)

**Stage별 가중치:**

| 기준 | Stage 0 | Stage 1 | Stage 2-3 |
|------|:-------:|:-------:|:---------:|
| 기술력 | 30% | 35% | 40% |
| 가격 | 25% | 20% | 15% |
| 납기 준수 | 20% | 25% | 30% |
| 소통 | 15% | 10% | 5% |
| 품질 | 10% | 10% | 10% |

**가중 평균 계산:**

```
가중 평균 = (기술력 × 가중치) + (가격 × 가중치) + ... + (품질 × 가중치)
```

**예시 (Stage 0, JLCPCB):**
- 기술력: 4점 (30%) = 1.2
- 가격: 5점 (25%) = 1.25
- 납기 준수: 5점 (20%) = 1.0
- 소통: 3점 (15%) = 0.45
- 품질: 4점 (10%) = 0.4
- **가중 평균**: 4.3점
