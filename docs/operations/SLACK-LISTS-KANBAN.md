## 5. Slack Lists 설정 가이드

### 5.1 List 생성

1. `#ggpnotice` 채널 접속
2. 채널 상단 **Lists** 탭 클릭
3. **+ Create List** 클릭
4. 이름: `EBS 업체 관리`

### 5.2 Custom Fields 추가

```
Lists 설정 → Custom Fields → Add Field

1. 상태 (Select)
   - 🟢 1순위
   - ⚠️ 2순위
   - 🔴 제외

2. 카테고리 (Select)
   - PCB
   - RFID
   - 조립
   - 통합

3. 적용 Stage (Multi-Select)
   - Stage 0
   - Stage 1
   - Stage 2
   - Stage 3

4. 예상 비용 (Text)
5. 담당자 (Person)
6. 마감일 (Date)
7. 평가점수 (Number)
8. 비고 (Text)
```

### 5.3 보기 설정

```
View → Board

Group by: 상태 (Status) 또는 컬럼 (Stage)
Sort by: 평가점수 (내림차순)
```

### 5.4 자동화 (Workflow Builder)

| 트리거 | 액션 |
|--------|------|
| 카드가 "견적 요청"으로 이동 | 담당자에게 DM 알림 |
| 마감일 D-3 | 채널에 리마인더 |
| 카드가 "제외"로 이동 | 비고 필드 입력 요청 |

## 6. 운영 규칙

### 6.1 카드 이동 규칙

| 이동 | 필수 조건 |
|------|----------|
| 후보 → 견적 요청 | 이메일 발송 완료 |
| 견적 요청 → 견적 수령 | 견적서 파일 첨부 |
| 견적 수령 → 협상 중 | 평가점수 4.0 이상 |
| 협상 중 → 계약 완료 | 계약서 체결 |
| * → 제외 | 비고에 사유 필수 |

| 협상 중 | 2 | 집중 협상 |
