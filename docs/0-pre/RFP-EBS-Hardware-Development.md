## 1. 프로젝트 개요

### 1.1 프로젝트 소개

**EBS (Event Broadcasting System)**는 RFID 기반 포커 카드 인식 시스템으로, 실시간 방송 오버레이 및 데이터 수집을 위한 하드웨어/소프트웨어 통합 솔루션입니다.

| 항목 | 내용 |
|------|------|
| **프로젝트명** | EBS (Event Broadcasting System) |
| **목적** | 포커 방송용 실시간 카드 인식 및 데이터 시스템 |
| **최종 목표** | WSOP STUDIO DB 자산 내재화 + 방송 워크플로우 자동화 |
| **개발 기간** | 2026-2028 (3개년) |
| **현재 단계** | Stage 0 (RFID 연결 검증) |

### 1.2 시스템 목적

1. **실시간 카드 인식**: RFID 태그가 내장된 포커 카드를 비접촉 방식으로 인식
2. **방송 오버레이**: 인식된 카드 정보를 실시간으로 방송 그래픽에 표시
3. **데이터 축적**: 핸드 히스토리, 플레이어 통계 등 DB 자산 확보
4. **운영 자동화**: 수동 입력 최소화를 통한 운영 효율성 향상

### 1.3 기술 스택 개요

## 3. 인터페이스 정의

### 3.1 MCU ↔ RFID 모듈 (SPI)

#### 3.1.1 핀아웃 다이어그램

```
┌─────────────────────────────────────────────────────────────────┐
│              ESP32 ↔ RFID Module SPI 연결                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ESP32 DevKit                      RFID Module (ST25R3911B)   │
│   ┌─────────────┐                   ┌─────────────┐            │
│   │             │                   │             │            │
│   │    3.3V  ●──┼───────────────────┼──● VCC      │            │
│   │             │     (빨강선)      │             │            │
│   │     GND  ●──┼───────────────────┼──● GND      │            │
│   │             │     (검정선)      │             │            │
│   │   GPIO5  ●──┼───────────────────┼──● SDA (SS) │            │
│   │             │     (노랑선)      │             │            │
│   │  GPIO18  ●──┼───────────────────┼──● SCK      │            │
│   │             │     (초록선)      │             │            │
│   │  GPIO23  ●──┼───────────────────┼──● MOSI     │            │
│   │             │     (파랑선)      │             │            │
│   │  GPIO19  ●──┼───────────────────┼──● MISO     │            │
│   │             │     (주황선)      │             │            │
│   │   GPIO4  ●──┼───────────────────┼──● RST      │            │
│   │             │     (흰색선)      │             │            │
│   │             │                   │     IRQ ●   │ (미사용)   │
│   └─────────────┘                   └─────────────┘            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 3.1.2 핀 매핑 상세

| ESP32 Pin | 기능 | RFID Pin | 설명 |
|-----------|------|----------|------|
| 3.3V | 전원 | VCC | 3.3V DC |
| GND | 접지 | GND | 공통 접지 |
| GPIO5 | SPI SS | SDA | 칩 선택 (Chip Select) |
| GPIO18 | SPI SCK | SCK | SPI 클럭 |
| GPIO23 | SPI MOSI | MOSI | Master Out Slave In |
| GPIO19 | SPI MISO | MISO | Master In Slave Out |
| GPIO4 | 리셋 | RST | 모듈 리셋 |
| - | 인터럽트 | IRQ | 미사용 (폴링 방식) |

#### 3.1.3 SPI 설정

| 파라미터 | 값 | 비고 |
|----------|------|------|
| SPI Mode | Mode 0 (CPOL=0, CPHA=0) | 표준 |
| Clock Speed | 1-10 MHz | 권장 4MHz |
| Bit Order | MSB First | 표준 |
| Data Width | 8-bit | 표준 |

#### 3.2.2 JSON 프로토콜 (ESP32 → Server)

```json

  "uid": "04:A2:B3:C4",
  "reader_id": 0,
  "timestamp": 1706832000000
```

```json
{
  "type": "error",
  "code": "RFID_TIMEOUT",

```
