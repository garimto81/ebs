## 3. Requirements

### 3.1 Functional Requirements

| ID | Requirement | Priority |
|----|-------------|:--------:|
| F1 | Read RFID tags in playing cards | Must |
| F2 | Detect 2 cards simultaneously | Must |
| F3 | Detection within 1 second | Must |
| F4 | Continuous operation for 8+ hours | Must |
| F5 | Support 52-card deck (unique ID per card) | Must |
| F6 | USB or Serial interface | Must |
| F7 | Support multiple readers (up to 10) | Should |

### 3.2 Physical Requirements

| ID | Requirement | Priority |
|----|-------------|:--------:|
| P1 | Compact form factor | Must |
| P2 | Max size: 150mm x 100mm x 20mm | Should |
| P3 | Durable construction | Must |

### 3.3 Technical Requirements

| ID | Requirement | Priority |
|----|-------------|:--------:|
| T1 | HF RFID (13.56 MHz) | Must |
| T2 | ISO 15693 or ISO 14443 | Must |
| T3 | SDK/API for Windows | Should |
| T4 | Read range: 3-5 cm | Should |
